# Terraform with AWS

**Tutorial basis:** this lab follows the progression of Chapter 2, Getting Started with
Terraform, moving from AWS/IAM setup, to Terraform installation, to a single EC2
instance, a single web server, a configurable deployment, an Auto Scaling Group, an
Application Load Balancer, and cleanup.

------

**By the end of this lab, students should be able to:**

 - 1. Explain the Terraform workflow: init, validate, plan, apply, and destroy.
 - 2. Configure the AWS provider and authenticate Terraform securely.
 - 3. Create AWS infrastructure declaratively using Terraform.
 - 4. Use variables, references, outputs, data sources, and lifecycle settings.
 - 5. Deploy a simple web service on AWS EC2.
 - 6. Deploy a small self-healing web tier using an Auto Scaling Group and an
Application Load Balancer.
 - 7. Clean up infrastructure safely to avoid unnecessary cloud costs
